ğŸ’¡ HOW CLAUDE EXTRACTS INSIGHTS

For each relevant speaker:

1. Get their transcript (first 8000 chars)
2. Send to Claude with problem context
3. Ask Claude to find relevant quote
4. Extract frameworks and concepts
5. Parse JSON response
6. Cache for future use

Cost: ~$0.0015 per query

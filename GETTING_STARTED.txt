TROUBLESHOOTING
===============

Q: "Command not found: python"
A: Make sure Python 3.9+ is installed
   python --version
   (Use python3 if needed)

Q: "No module named fastapi"
A: Install dependencies
   pip install -r backend/requirements.txt

Q: "Connection refused on port 8000"
A: Backend not running
   cd backend && python main.py

Q: "results show placeholder data"
A: Using static version (wwld.html)
   Use live version instead (frontend_backend_integration.html)
   And make sure backend is running
